# Changelog

## [1.0.0] - 2024-11-12

### Added
- ✅ Quản lý thông tin dự án Lean Six Sigma
- ✅ Quản lý thành viên và stakeholders
- ✅ Kế hoạch chi tiết với Gantt chart (DMAIC phases)
- ✅ Bảng ký tên/phê duyệt
- ✅ Dashboard thống kê với nhiều loại biểu đồ
- ✅ Xuất báo cáo PDF, Excel, CSV
- ✅ Quản lý danh mục Phòng/Ban
- ✅ Import/Export dữ liệu
- ✅ Hướng dẫn sử dụng chi tiết
- ✅ Hỗ trợ 5 danh mục dự án Lean:
  - An toàn người bệnh
  - Hài lòng người bệnh
  - Hài lòng nhân viên
  - Nâng cao chất lượng chuyên môn
  - Bệnh viện thông minh

### Features
- SQLite database để lưu trữ dữ liệu
- Giao diện Streamlit thân thiện, responsive
- Biểu đồ Gantt tương tác với Plotly
- Báo cáo PDF hỗ trợ tiếng Việt
- Dashboard tùy chỉnh với nhiều loại chart
- Cảnh báo công việc quá hạn
- Tính toán tiến độ tự động

### Technical
- Python 3.8+
- Streamlit 1.31.0
- SQLite database
- Plotly cho visualization
- ReportLab cho PDF export
- pandas cho data manipulation
