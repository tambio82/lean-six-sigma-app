- Rate limiting

- User authentication required
- Edit/delete own comments
- Admin moderation tools (future)
- Comment guidelines

### Risk 4: Performance with Many Activities

**Mitigation:**
- Pagination (50 per page)
- Date range filters
- Database indexes
- Archive old activities
